CYCOD CONFIG SET

  Set the value of a configuration setting.

USAGE: cycod config set KEY VALUE [--global | --user | --local]

ARGUMENTS

  KEY              The name of the configuration setting
  VALUE            The value to set

OPTIONS

  SCOPE OPTIONS

    --global, -g    Use global scope (all users)
    --user, -u      Use user scope (current user)
    --local, -l     Use local scope (default)

EXAMPLES

  EXAMPLE 1: Automatically approve read-only function calls

    cycod config set App.AutoApprove read

  EXAMPLE 2: Set the default OpenAI model in local configuration

    cycod config set OPENAI_CHAT_MODEL_NAME gpt-5-mini

  EXAMPLE 3: Set the OpenAI API key in user configuration

    cycod config set OPENAI_API_KEY sk-123-eg-your-key-xyz --user

  EXAMPLE 4: Set the Azure OpenAI endpoint for all users (global scope)

    cycod config set AZURE_OPENAI_ENDPOINT "https://your-resource.openai.azure.com" --global
