CYCOD MCP ADD

  Adds a new Model Context Protocol (MCP) server registration.
  
USAGE: cycod mcp add SERVER_NAME --command "COMMAND ARGS" [...]
   OR: cycod mcp add SERVER_NAME --url URL [...]

  Where SERVER_NAME is the name for the new MCP server registration.

OPTIONS

  STDIO SERVERS

    --command       The command to execute to start the MCP server (for STDIO transport)
    --arg ARG       An argument to pass to the command (can be used multiple times)
    --args "ARGS"   Space-separated arguments to pass to the command (alternative to using multiple --arg)
    --env, -e       An environment variable in KEY=VALUE format (can be used multiple times)

  SSE SERVERS

    --url           The URL for the SSE endpoint (for SSE transport)

  SCOPE OPTIONS

    --global, -g    Add registration to global scope (all users)
    --user, -u      Add registration to user scope (current user)
    --local, -l     Add registration to local scope (current directory, default)

EXAMPLES

  EXAMPLE 1: Register Playwright MCP for web automation

    cycod mcp add playwright --command "npx" --args "@playwright/mcp@latest"

  EXAMPLE 2: Register Playwright MCP in headless mode (for servers)

    cycod mcp add playwright-headless --command "npx" --args "@playwright/mcp@latest --headless"

  EXAMPLE 3: Register Playwright MCP with Chrome browser

    cycod mcp add playwright-chrome --command "npx" --args "@playwright/mcp@latest --browser chrome"

  EXAMPLE 4: Register Playwright MCP with advanced capabilities

    cycod mcp add playwright-full --command "npx" --args "@playwright/mcp@latest --caps vision,pdf"

  EXAMPLE 5: Register a remote MCP server using SSE

    cycod mcp add remote-mcp --url "https://mcp-server.example.com/mcp"

  EXAMPLE 6: Register a local MCP server with specific scope

    cycod mcp add local-playwright --command "npx" --args "@playwright/mcp@latest" --user
