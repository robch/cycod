CYCOD TOOL REMOVE

  Remove a custom tool.

USAGE: cycod tool remove TOOL_NAME [options]

ARGUMENTS

  TOOL_NAME         Name of the tool to remove

OPTIONS

  SCOPE OPTIONS

    --global, -g    Remove tool from global scope (all users)
    --user, -u      Remove tool from user scope (current user)
    --local, -l     Remove tool from local scope (current project, default)
    --any, -a       Remove tool from first scope where found

EXAMPLES

  EXAMPLE 1: Remove a tool from local scope
    cycod tool remove git-push-all

  EXAMPLE 2: Remove a tool from global scope
    cycod tool remove search-code --global
