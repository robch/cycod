USE APP SETTINGS

  These settings control CycoD's application behavior, including auto-saving,
  function call permissions, and API timeouts.

  MAX PROMPT TOKENS

    EXAMPLE 1: Set maximum prompt tokens via config

      cycod config set App.MaxPromptTokens 50000
      cycod config set App.MaxPromptTokens 50000 --local
      cycod config set App.MaxPromptTokens 50000 --user
      cycod config set App.MaxPromptTokens 50000 --global

    EXAMPLE 2: Set maximum prompt tokens via environment variable

      # Bash (Linux/macOS)
      export CYCOD_MAX_PROMPT_TOKENS="50000"

      # CMD (Windows)
      set CYCOD_MAX_PROMPT_TOKENS=50000

      # PowerShell (Windows)
      $env:CYCOD_MAX_PROMPT_TOKENS = "50000"

    EXAMPLE 3: Supply maximum prompt tokens via command line

      cycod chat --max-prompt-tokens 50000

  AI-BASED CHAT COMPACTION

    EXAMPLE 1: Configure AI-based compaction for chat history

      # Enable full (detailed) compaction mode
      cycod config set App.CompactionMode full

      # Enable simple compaction mode
      cycod config set App.CompactionMode simple

      # Disable compaction
      cycod config set App.CompactionMode none

      # Set locally, for user, or globally
      cycod config set App.CompactionMode full --local
      cycod config set App.CompactionMode full --user
      cycod config set App.CompactionMode full --global

    EXAMPLE 2: Configure preservation of recent messages

      # Set number of recent messages to preserve intact
      cycod config set App.PreserveMessages 6 --user

    EXAMPLE 3: Control compaction via environment variables

      # Bash (Linux/macOS)
      export CYCOD_COMPACTION_MODE="full"
      export CYCOD_PRESERVE_MESSAGES="6"

      # PowerShell (Windows)
      $env:CYCOD_COMPACTION_MODE = "full"
      $env:CYCOD_PRESERVE_MESSAGES = "6"

    EXAMPLE 4: Configure compaction via command line

      # Enable full compaction mode
      cycod chat --compact

      # Specify compaction mode
      cycod chat --compact full
      cycod chat --compact simple
      
      # Disable compaction
      cycod chat --no-compact
      cycod chat --compact none
