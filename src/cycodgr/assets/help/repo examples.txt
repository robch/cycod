CYCODGR REPO COMMAND EXAMPLES

USAGE: cycodgr repo <keywords...> [options]

EXAMPLES

  EXAMPLE 1: Find specific .NET libraries

    cycodgr repo "minimal api" --language csharp --min-stars 500
    cycodgr repo "MediatR" --owner jbogard
    cycodgr repo "Polly retry" --language csharp

  EXAMPLE 2: Find authentication/authorization libraries

    cycodgr repo "JWT authentication" --language csharp --min-stars 100
    cycodgr repo "OAuth2 client" --language javascript
    cycodgr repo "passport" --owner jaredhanson

  EXAMPLE 3: Find testing frameworks and tools

    cycodgr repo "xunit" --language csharp --min-stars 1000
    cycodgr repo "mocha" --language javascript --exclude-fork
    cycodgr repo "pytest" --language python --owner pytest-dev

  EXAMPLE 4: Find CI/CD and DevOps tools

    cycodgr repo "github actions" --owner actions
    cycodgr repo "terraform provider" --language go --min-stars 100
    cycodgr repo "kubernetes operator" --language go

  EXAMPLE 5: Find database libraries and ORMs

    cycodgr repo "Entity Framework" --owner dotnet
    cycodgr repo "Dapper" --language csharp
    cycodgr repo "Sequelize" --language javascript

  EXAMPLE 6: Clone popular example repositories

    cycodgr repo "clean architecture" --language csharp --clone --max-clone 3
    cycodgr repo "microservices example" --owner dotnet --clone --clone-dir ./examples
    cycodgr repo "design patterns" --language csharp --clone --as-submodules

  EXAMPLE 7: Find UI component libraries

    cycodgr repo "headless ui" --owner tailwindlabs
    cycodgr repo "shadcn ui" --language typescript
    cycodgr repo "material ui" --owner mui

  EXAMPLE 8: Save and export search results

    cycodgr repo "rate limiting" --language csharp --save-json rate-limit-libs.json
    cycodgr repo "logging framework" --save-csv logging-libs.csv --save-urls urls.txt
    cycodgr repo "caching" --language csharp --save-table cache-libs.md

  EXAMPLE 9: Combine multiple filters

    cycodgr repo "validation library" --owner FluentValidation --language csharp --min-stars 500 --exclude-fork
    cycodgr repo "api client" --language typescript --min-stars 1000 --format table

  EXAMPLE 10: Exclude certain patterns

    cycodgr repo "dotnet template" --exclude "archived" "deprecated"
    cycodgr repo "starter kit" --language csharp --exclude "fork"
