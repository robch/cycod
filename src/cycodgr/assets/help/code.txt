CYCODGR CODE COMMAND

Search for code within GitHub repositories by content and file type.

USAGE: cycodgr code <keywords...> [options]

DESCRIPTION

  The code command searches GitHub for code files containing your keywords.
  Results show matching lines with context, formatted in Markdown with code
  fences and line numbers. You can filter by file extension, repository,
  language, and more.

EXAMPLES

  EXAMPLE 1: Find AI agent instruction files and patterns

    cycodgr code "AGENTS.md" --extension md
    cycodgr code "AI Instructions" --extension md --owner microsoft

  EXAMPLE 2: Find authentication implementations

    cycodgr code "AddJwtBearer" --extension cs
    cycodgr code "passport.authenticate" --extension js

  EXAMPLE 3: Find GitHub Actions workflows and config

    cycodgr code "actions/checkout" --extension yml --owner microsoft --lines 10
    cycodgr code "appsettings.json" --owner dotnet --lines 5
