CYCODGR - GitHub Repository and Code Search

USAGE: cycodgr repo <keywords...> [options]
   OR: cycodgr code <keywords...> [options]

DESCRIPTION

  cycodgr is a command-line tool for searching GitHub repositories and code.
  It provides two main commands:
  
  - repo: Search for repositories by keywords, topics, or descriptions
  - code: Search for code within repositories by file content
  
  Results can be filtered by language, owner, stars, file extension, and more.

QUICK START

  Search for repositories:
    cycodgr repo "dotnet aspire"
    cycodgr repo "ai agents" --owner microsoft

  Search for code:
    cycodgr code "TODO" --extension cs
    cycodgr code "AGENTS.md" --repo microsoft/vscode

COMMON OPTIONS

  --max-results N            Maximum search results
  --language LANG            Filter by programming language
  --owner ORG                Filter by owner/organization
  --min-stars N              Filter by minimum stars
  --format FORMAT            Output format (detailed, url, table, json, csv)
