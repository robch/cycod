The 'ama' (Ask Me Anything) feature allows you to ask natural language questions about how to use the cycodgr tool.

USAGE: cycodgr help ama "your question here"

EXAMPLES:
  cycodgr help ama "How do I search for repositories by organization?"
  cycodgr help ama "How can I search code within specific repositories?"
  cycodgr help ama "What output formats are available?"

This feature combines AI with the comprehensive help documentation to provide helpful answers to your questions.
