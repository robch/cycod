CYCODGR REPO COMMAND

Search for GitHub repositories by keywords, topics, or descriptions.

USAGE: cycodgr repo <keywords...> [options]

DESCRIPTION

  The repo command searches GitHub for repositories matching your keywords.
  Results can be filtered by language, owner, stars, and more. You can also
  clone the results directly to your local machine.

EXAMPLES

  EXAMPLE 1: Find popular .NET libraries

    cycodgr repo "minimal api" --language csharp --min-stars 500
    cycodgr repo "rate limiting" --owner dotnet

  EXAMPLE 2: Clone React component libraries

    cycodgr repo "headless ui components" --language typescript --clone --max-clone 3

  EXAMPLE 3: Find and save Python ML frameworks

    cycodgr repo "machine learning framework" --language python --min-stars 10000 --save-json ml-frameworks.json
