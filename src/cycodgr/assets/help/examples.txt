USAGE: cycodgr repo <keywords...> [options]
   OR: cycodgr code <keywords...> [options]

EXAMPLES

  EXAMPLE 1: Find popular .NET libraries

    cycodgr repo "minimal api" --language csharp --min-stars 500
    cycodgr repo "mediator pattern" --owner dotnet --min-stars 1000

  EXAMPLE 2: Search for React component libraries

    cycodgr repo "headless ui components" --language typescript
    cycodgr code "use client" --extension tsx --owner vercel --lines 5

  EXAMPLE 3: Find authentication implementations

    cycodgr code "JwtBearer" --extension cs --min-stars 100
    cycodgr code "passport.authenticate" --extension js --repo expressjs/express

  EXAMPLE 4: Find GitHub Actions workflows

    cycodgr code "uses: actions/checkout" --extension yml --owner microsoft
    cycodgr code "runs-on: ubuntu-latest" --in-files yml --format filenames

  EXAMPLE 5: Clone and study popular repos

    cycodgr repo "rate limiting middleware" --language csharp --clone --max-clone 3
    cycodgr repo "retry policy" --owner App-vNext --save-json polly-repos.json

  EXAMPLE 6: Find configuration examples

    cycodgr code "Serilog.Settings" --extension json --lines 10
    cycodgr code "appsettings.json" --repo dotnet/aspire --owner dotnet
