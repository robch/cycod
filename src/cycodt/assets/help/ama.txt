The 'ama' (Ask Me Anything) feature allows you to ask natural language questions about how to use the cycod, cycodmd, and cycodt tools.

USAGE: cycodt help ama "your question here"

EXAMPLES:
  cycodt help ama "How do I run tests that contain specific text?"
  cycodt help ama "How do I use regex patterns to check command output?"
  cycodt help ama "How do I save test output in TRX or JUnit format?"

This feature combines AI with the comprehensive help documentation to provide helpful answers to your questions.
