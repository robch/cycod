CYCODT LIST

  The cycodt list command lists CLI YAML tests.

USAGE: cycodt list [...]

  FILES
    --file FILE
    --files FILE1 [FILE2 [...]]
    --files PATTERN1 [PATTERN2 [...]]
    --exclude-files PATTERN1 [PATTERN2 [...]]

  TESTS
    --test TEXT
    --tests TEXT1 [TEXT2 [...]]
    --contains TEXT1 [TEXT2 [...]]
    --remove TEXT1 [TEXT2 [...]]
